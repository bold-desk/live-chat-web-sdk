# BoldDesk Live Chat Web SDK

The **BoldDesk Live Chat Web SDK** enables seamless integration of the BoldDesk live chat widget into web applications built with frameworks like **React**, **Vue**, or **Angular**. It wraps native widget functionalities and provides full TypeScript support to streamline your development process.
